# Kutt

Kutt is a modern open-source URL shortener. It supports custom domains, link expiration, password protection, basic analytics, and a clean web interface.

## Features

- Custom domains for branded short links
- Link expiration and password protection
- REST API for programmatic shortening
- Visit statistics and basic analytics
- Editable destinations

## Repository

https://github.com/thedevs-network/kutt

## Default Ports

- Web UI / API: 3000
- Postgres: 5432

Configure environment variables (e.g. DB connections, JWT secret, default domain) via your platform before running in production.
